Integração com API de Maquiagem

Exemplos para serem usados:
Marca: Covergirl
Tipo de produto: blush
Preço Maior Que: 10

Pode-se encontrar a API em https://makeup-api.herokuapp.com/

Visão Geral do Projeto:

  • Interface HTML Responsiva: com layout construído no Bootstrap.
  • Apresentação de dados da API: incluindo marca, tipo de produto e preço.
  • Documentação da API: com link para mais informações.
  • Integração com 3 métodos da API: para pesquisa por marca, tipo de produto e preço, utilizando funções JavaScript.
  • Exibição de resultados: em uma lista sem duplicatas.
  • Permite pesquisas: amplas ou filtradas por marca, tipo e preço.
